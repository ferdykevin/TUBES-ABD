# Tugas Besar Analisis Big Data (TUBES ABD)

Repository ini digunakan untuk menyimpan seluruh source code, skrip pemrosesan data, dan dokumentasi Tugas Besar mata kuliah Analisis Big Data. Proyek ini berfokus pada implementasi, pengelolaan, serta analisis data skala besar menggunakan ekosistem Big Data.

## Deskripsi Proyek
Proyek ini dirancang untuk menangani pemrosesan dan analisis data berskala besar guna menyelesaikan studi kasus tertentu. Fokus utamanya adalah memanfaatkan arsitektur penyimpanan terdistribusi dan framework pemrosesan data untuk mengeksplorasi, mentransformasi, serta menghasilkan insight yang relevan dari dataset yang digunakan.

---

## Fitur Utama
* Pemrosesan Data Terdistribusi: Mengelola dan memproses dataset berukuran besar secara efisien memanfaatkan arsitektur Big Data.
* Analisis dan Query Data: Implementasi skrip pemrosesan data untuk kebutuhan pembersihan, transformasi, dan penarikan informasi penting.
* Visualisasi Insight: Menyajikan hasil pemrosesan data ke dalam bentuk laporan atau dashboard interaktif yang mudah dipahami.

## Teknologi yang Digunakan
